
# grow

  Grow textareas without using a clone or ghost element.

  Based off of: https://github.com/javierjulio/textarea-autosize.

## Example

```html
<textarea id="content"></textarea>
```

```js
grow(content);
```

```css
#content {
  box-sizing: border-box;
  max-height: 94px; /* optional */
  min-height: 31px;
}
```
